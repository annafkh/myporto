Most recent work

Employee Performance Evaluation System

The Employee Performance Evaluation System is a web-based application designed to help organizations evaluate and monitor employee performance objectively and efficiently. The system implements the Key Performance Indicator (KPI) method to assess individual achievements based on predefined categories and indicators. It supports multiple user roles such as Human Capital (HC), Leader, and Employee, each with different access levels and responsibilities.

GitHub repository

Caraka Auto Service - Booking Online Automotive

This project is an online booking system for the Caraka Auto Service Workshop which allows customers to book services digitally. This platform makes it easier for customers to choose the type of service, determine the service schedule, and get real-time confirmation. With database integration and automatic notifications, this system aims to increase workshop operational efficiency and customer comfort.

GitHub repository

Abule - Adopt and Donate Animals

This project is a digital platform for Adopt and Donate Pets, which allows users to search for, adopt or give away pets easily. This system provides search features, animal data management, and communication between adopters and owners. The backend is built using Express.js, ensuring fast, secure, and scalable performance to handle user requests efficiently.

GitHub repository

Car Rental UI/UX

This project is a UI/UX design for a car rental platform, focusing on creating an intuitive and seamless user experience. It enables users to search for, book, and manage car rentals effortlessly. The design includes features such as vehicle search, booking management, payment integration, and user authentication. The goal is to ensure a user-friendly interface with a visually appealing and responsive design for both web and mobile platforms.
